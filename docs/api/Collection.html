registerIds(newIds, guid)

Source:
  • mixins/cacheMixin.js, line 306
Mixes In:
  • cacheMixin.registerIds
Registers a list of Ids that a particular object cares about and pushes any cached models its way. This method intelligently updates the "_requestedIds" field to contain all unique requests for Ids to be fetched. Furthermore, the "polledFetch" method is overriden such that it no longer routes through Backbone's fetch all, but rather a custom "fetchByIds" method.
Parameters:
Name Type Description
newIds Array New ids to register under the requester
guid string The GUID of the object that wants the ids
