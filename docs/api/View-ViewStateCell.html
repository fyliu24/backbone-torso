View~ ViewStateCell

new ViewStateCell(attrs, opts)

Source:
  • View.js, line 34
ViewStateCell is a NestedCell that holds view state data and can trigger change events. These changes events will propogate up and trigger on the view as well.
Parameters:
Name Type Description
attrs Object the initial values to set on the cell - inherited from NestedCell.
opts Object options for the cell.
Properties
Name Type Description
view external:Backbone-View the view that these options are tied to.
